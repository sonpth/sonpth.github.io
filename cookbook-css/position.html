Position

The default behavior of a <div> is taking all the available width, and the height to accommodate its content if required.
position: fixed
This element is positioned in the document regardless its parents.
position: absolute removes the element from the webpage flow. It is positioned according to the coordinate system it is in (i.e default or the closest parent with a set positioning).
  • Its container and the lexically following elements won't be affected by its existence.
  • Like relative, it creates a new coordinate system for its child elements.
  • By specifing all top, left, right and bottom, we actually define the element size depending on its container (scalable)
